## 1. 選取 Songs 資料夾位置
通常在你的 osu! 遊戲資料夾內。

> 可以在 settings 頁面看到所選的路徑
---

## 2. 選歌並加入待壓縮清單
在左側列表選擇歌曲，然後按下 **"→ 加入"**，選中的歌曲就會跑到右側清單。

> 如果想從右側清單移除歌曲，選取後按 **"← 移除"** 即可。
---

## 3. 選擇輸出資料夾
選擇你想要儲存 `.osz` 檔案的位置。

> 可以在 settings 頁面看到所選的路徑
---

## 4. 開始壓縮
按下 **"開始壓縮"**，等待進度完成就好啦!
